## 🪙 **EGILI - "IL CARBURANTE DELL'ECOSISTEMA"**

### **CONCEPT:**

```
💎 Token Utility Interno - NON è una cryptocurrency!
├─ NON trasferibile tra utenti (account-bound)
├─ NON scambiabile su exchange esterni
├─ Guadagnato via merit + attività piattaforma
├─ Spendibile SOLO in-platform (utility pura)
├─ Scadenza: 12 mesi (earned) / ∞ (purchased in packs)
└─ Purpose: Fee reduction + Services + Gamification
```

**Conversion Rate Base:** `1 Egilo = €0.10` (perceived value)

**Anti-Speculazione Compliance:**

-   Trattati come "Loyalty Points" (simile a Punti Esselunga)
-   Nessuna tassazione all'accumulo per l'utente
-   Non costituiscono "moneta elettronica" (no cash-out)
-   T&C espliciti: "Egili have no monetary value outside platform"

**B) Primary Market Fee Discount (LIMITATO):**

```
Fee standard primario: ~30% (varia per volume globale)

Con Egili discount:
├─ Spendi 150 Egili → -20% sulla fee (max)
├─ Esempio: Fee €300 → €240 (risparmio €60)
└─ Limiti:
    - Max -20% (vs -80% secondario)
    - Solo per vendite < €5k (anti-abuse whales)
    - Solo una volta per EGI (al mint/prima vendita)

Reason: Incentivare uso Egili ma non cannibalizzare
        revenue primario (che è core business)
```

**NOTA IMPORTANTE:**

```
⚠️ Equilibrium vs Egili:
├─ Equilibrium accoppiato = 0% fee AUTOMATICA (priorità)
├─ Egili = sconto fee su transazioni SENZA Equilibrium
└─ Non combinabili (o Equilibrium o Egili, non entrambi)

Logica:
- EGI con Equilibrium → sempre 0% fee (benefit permanente)
- EGI senza Equilibrium → può usare Egili per sconto
```
